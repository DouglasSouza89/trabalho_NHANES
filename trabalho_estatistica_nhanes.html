4. Distribuio Binomial no NHANES

Neste item, foi utilizada a varivel binria JaFumou100Cigarros, que indica se o indivduo j fumou pelo menos 100 cigarros na vida. A partir dessa varivel, foi calculada a probabilidade emprica p. Em seguida, foram geradas e comparadas graficamente as distribuies Binomial(n = 100, p = p) e Poisson( = np).

In[14]:

# Mantendo apenas valores vlidos: 1 = Sim, 2 = No

# Convertendo para binrio: 1 = Sim, 0 = No
fumou_bin = (fumou == 1).astype(int)

# Probabilidade emprica p-hat
p_hat = fumou_bin.mean()

In[15]:
from scipy.stats import binom, poisson

n = 100
lam = n * p_hat

k = np.arange(0, n + 1)

pmf_binom = binom.pmf(k, n, p_hat)
pmf_pois = poisson.pmf(k, lam)

plt.figure(figsize=(10, 6))

plt.stem(k, pmf_binom, basefmt=" ", label="Binomial(100, p)")
plt.stem(k, pmf_pois, basefmt=" ", markerfmt="C1o", linefmt="C1-", label=f"Poisson(={lam:.2f})")

plt.xlim(max(0, int(lam - 4*np.sqrt(lam))), int(lam + 4*np.sqrt(lam)))
plt.xlabel("Nmero de fumantes em 100 pessoas (k)")
plt.ylabel("PMF")
plt.title("Comparao entre Binomial e Poisson")
plt.legend()
plt.tight_layout()
plt.show()

Interpretao

A probabilidade emprica p representa a proporo de indivduos que j fumaram pelo menos 100 cigarros na amostra. A distribuio Binomial modela o nmero esperado de fumantes em 100 pessoas. A distribuio de Poisson, com = np, fornece uma boa aproximao da Binomial para esse cenrio, o que pode ser visualizado pela forte semelhana entre as duas curvas.

Isso confirma que variveis positivas com cauda longa podem ser bem modeladas por uma distribuio log-normal, conforme discutido nos slides da disciplina.
